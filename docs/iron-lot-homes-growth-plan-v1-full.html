Section 05

Pre-Launch Checklist

Everything that needs to happen to take your site from demo to live, categorized by priority. The critical items must be completed before launch.

Critical — Required for Launch
  • 1
    Connect custom domain. Currently serving from GitHub Pages. Need to connect ironlothomes.com, configure DNS, and verify ownership. Without this, the site remains invisible to customers.
  • 2
    Submit to Google Search Console. Site isn't indexed by Google yet. Need to verify domain, submit sitemap, and request indexing for all 36 pages.
  • 3
    Configure form handling. Replace placeholder API keys with real Web3Forms keys across all 36 pages so leads flow into your inbox immediately upon launch.
  • 4
    Install analytics and tracking. Set up Google Analytics 4, Google Tag Manager, and conversion tracking so you can measure traffic and ROI from day one.
High Priority — Week 1-2 Post-Launch
  • 5
    Create Google Business Profile. Essential for appearing in local 3-pack searches ("mobile home buyers near me"). This is how you capture high-intent local searches.
  • 6
    Add privacy policy page. Required for legal compliance since forms collect personal data (name, phone, address). Builds trust with potential sellers.
  • 7
    Fix OG image paths. Social sharing images reference GitHub domain. Need to update to ironlothomes.com so link previews work correctly on Facebook, iMessage, etc.
  • 8
    Remove SearchAction schema. References a search page that doesn't exist on your static site. Clean this up to avoid sending incorrect signals to Google.
  • 9
    Create /locations/ index page. Breadcrumb schema references a location hub page. Building this creates better internal linking and user navigation.
Medium Priority — Month 1-2 Optimization
  • 10
    Replace stock photography. All images are stock photos from Unsplash. Real mobile home photos improve trust and enable image SEO optimization.
  • 11
    Fix duplicate AggregateRating data. Same review count (52 reviews) on every location page. Should reflect actual reviews per location or remove per-city ratings.
  • 12
    Add social media profiles to schema. Currently empty sameAs array. Link Facebook, Instagram profiles for stronger brand signals to Google.
  • 13
    Build blog/content hub. Capture long-tail keyword traffic for searches like "how to sell a mobile home in Texas" and "mobile home owner financing."
  • 14
    Create inventory/listings page. Buyers need a way to browse available mobile homes. This increases engagement and time-on-site.
  • 15
    Fix WhatsApp pre-fill text. Buy page WhatsApp link says "selling" instead of "buying" — minor but confusing for buyers.
